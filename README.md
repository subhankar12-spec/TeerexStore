# Teerex Store

A simple e commerce project challenge.

## Run Project :

npm i && npm start

### Tech Stacks
React
React Router Dom
API
MaterialUI
HTML
JavaScript
CSS

### Functionalities
Product Page with searching and filtering options
User can add product into cart
User can increase/decrease quantity of product and also delete it from the cart
